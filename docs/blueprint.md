# **App Name**: Astraea Intelligence

## Core Features:

- Live Signal Dashboard: A high-fidelity monitoring interface featuring real-time transaction streams with live behavioral risk overlays and animated anomaly trends.
- Graph Intelligence Hub: Utilizes Neo4j and multi-hop tracing to persist and analyze complex entity relationships and circular money laundering patterns.
- Autonomous Agent Investigator: An AI-powered tool that automatically constructs full investigation cases by reasoning across account relationships and network history.
- Interactive Network Vis: Advanced graph visualization engine built with Cytoscape.js for interactive account exploration, timeline replay, and cluster discovery.
- Conversational Compliance Copilot: An LLM-driven chat assistant that enables analysts to query graph evidence and account risk scores using natural language.
- Narrative Synthesis Tool: An generative AI engine that uses investigative data to tool automatically authored Suspicious Activity Reports (SARs) and evidence summaries.
- Audit Case Management: Enterprise-grade workflow engine for assigning investigations, tracking escalation statuses, and logging analyst feedback into a PostgreSQL store.

## Style Guidelines:

- Primary color: Electric Blue (#3D5AFE) for a high-performance, precision-focused technological vibe.
- Background color: Deep Midnight (#0A0C14) providing a professional dark-themed aesthetic typical of modern cybersecurity platforms.
- Accent color: Atmospheric Azure (#00B0FF) for differentiating primary navigation elements and subtle interface highlights.
- Headlines: 'Space Grotesk' for a computerized, techy feel; Body: 'Inter' for objective and clear readability; Code/Logs: 'Source Code Pro' for technical accuracy.
- A bento-style modular layout using glassmorphism principles, ensuring high information density is presented in a premium, organized structure.
- Sharp, thin-stroke monolinear icons that convey technical sophistication and align with enterprise regtech standards.
- Micro-interactions powered by Framer Motion, featuring smooth transitions for real-time risk fluctuations and subtle graph expansion effects.
